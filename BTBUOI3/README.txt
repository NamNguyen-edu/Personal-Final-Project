TDash Mini Store Manager (WinForms .NET 8)
Project created for user request (store name: tdash).

How to run:
1. Requires .NET 8 SDK and Visual Studio 2022 (or dotnet SDK + editor).
2. Open a terminal in the project folder and run: dotnet build
3. Run with: dotnet run
   or open the folder in Visual Studio and press F5.

Notes:
- Project uses System.Drawing and PrintDocument; run on Windows.
- UI files are created with Designer-style partial classes so you can open forms in Visual Studio Designer.
